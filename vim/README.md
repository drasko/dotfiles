## Vim Dotfiles
Uses [vim-plug](https://github.com/junegunn/vim-plug)

## Install
```
cp ./.vimrc ~/.vimrc
```

Now install [vim-plug](https://github.com/junegunn/vim-plug) plugins and [vim-go](https://github.com/fatih/vim-go) binaries. Enter vim and execute:
```
:PlugInstall
:GoInstallBinaries
```
